安装模块

Module :: install( string appid )

函数参数
参数 含义
string appid 模块ID
返回值
值 含义
string func:模块目录没有写权限
lock:模块已经安装
exec:没有找到模块安装脚本（install.sql）
stop:模块安装失败，SQL 出错
done:完成安装

卸载模块

Module :: uninstall( string appid )

函数参数
参数 含义
string appid 模块ID
返回值
值 含义
string lock:模块已经卸载
exec:没有找到模块卸载脚本（uninstall.sql）
stop:模块卸载失败，SQL 出错
done:完成卸载
