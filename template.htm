VeryIDE 通行证模板构成

VeryIDE 通行证中模板是以独立文件夹的形式存放于 module/passport/theme 中，其中每个文件夹对应一个模板

VeryIDE 通行证模板文件

images 用于存放模板的图片元素
js 用于存放模板前端脚本文件
style 用于存放模板CSS样式文件
header.php 模板头部，供 PHP 引用
header.htm 公共头部，供模板引用
footer.php 模板底部，供 PHP 引用
footer.htm 公共底部，供模板引用
* 其他资源文件

VeryIDE 通行证模板勾子

passport:naver 全局顶部导航条
passport:header 模块头
passport:footer 模块尾
form:view 表单（包括试题、辩论和投票）查看页
form:post 表单（包括试题、辩论和投票）提交页
