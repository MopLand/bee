VeryIDE 小工具构成

VeryIDE 中小工具可以看作是模块的扩展，以独立文件夹的形式存放于 module/*/widget/ 中，每个小工具的名字（我们称之为 widget ）都是唯一的。

config.xml

工具配置文件，以 xml 形式存储，以下相关常量：
name 工具名称，推荐4个字以内
width 宽度尺寸
height 高度尺寸
script 动态载入的脚本文件（例如：{WIDGET}widget.js）
x 相对于屏幕左边距位置
y 相对于屏幕顶边距位置
fx 相对于屏幕右边距位置
fy 相对于屏幕底边距位置
html 内容块
target 目标容器
选择器:具体容器中（请填写 CSS 选择器路径，如：#container）
[空]:桌面任意位置
replace 插入方式
false:具将会直接追加到容器中（默认值）
true:将会直接追加（innerHTML）到容器中
draggable 是否允许拖动
true:允许，可以随意拖动
false:不允许，固定在某处

config.xml <html> 节点关键字

在 html 节点内容块中，可以使用以下关键字来替换实际的内容：
{HOST} 系统绝对地址，如：http://localhost/Rainbow/
{MODULE} 当前模块目录，如：module/*/
{WIDGET} 当前工具目录，如：module/*/widget/*/
{APPID} 当前模块 ID
{WGTID} 当前工具 ID
