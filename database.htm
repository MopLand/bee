打开数据库

array System :: connect()

返回值
值 含义
array 返回的查询结果
数据库打开后，可以直接使用 System :: $db 进行操作 
//连接数据库
System :: connect();

//更新数据
System :: $db -> execute( "UPDATE `sys:admin` SET theme='".$theme."' WHERE id = ".$_G['manager']['id'] );

关闭数据库

array System :: connect()

返回值
值 含义
array 返回的查询结果 
//关闭数据库
System :: connect();

//更新数据
var_dump( System :: $db );
