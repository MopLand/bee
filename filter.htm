过滤特殊字符

string UnFilterHtml( string str )

函数参数
参数 含义
string str 过滤前的字符串
返回值
值 含义
string 过滤后的字符串
本接口函数用于过滤字符串中的空格，换行等特殊字符和html特殊标签
用户登录示例 (PHP)
$str="s<br />s";
echo UnFilterHtml($str); //ss

过滤html标签

string FilterHtml( string str )

函数参数
参数 含义
string str 过滤前的字符串
返回值
值 含义
string 过滤后的字符串
本接口函数用于过滤字符串中的html标签
用户登录示例 (PHP)
$str="s<br />s";
echo FilterHtml($str); //ss

过滤UBB标签

string clear_ubb( string str )

函数参数
参数 含义
string str 过滤前的字符串
返回值
值 含义
string 过滤后的字符串
本接口函数用于过滤字符串中的UBB标签
用户登录示例 (PHP)
$str="[b]ss[/b]";
echo clear_ubb($str); //ss

过滤html中的换行空格等

string DeleteHtml( string str )

函数参数
参数 含义
string str 过滤前的字符串
返回值
值 含义
string 过滤后的字符串
本接口函数用于过滤字符串中的\t,\n,\r\n,\r," "
用户登录示例 (PHP)
$str="ss\t";
echo DeleteHtml($str); //ss

压缩html

string compress( string str )

函数参数
参数 含义
string str 过滤前的字符串
返回值
值 含义
string 过滤后的字符串

本接口函数用于压缩HTML，删除无用空白（注意：以下形式的注释暂时不能处理）

<script><!--
--></script>

用户登录示例 (PHP)
$str="<br />";
echo compress($str); //<br/>

string inject_filter( string content )

函数参数
参数 含义
string str 过滤前的字符串
返回值
值 含义
string content 过滤后的字符串

本接口函数用于过滤全局输入,将全局的html、ubb编码以及特殊字符进行递归过滤

sql检测

Boolean inject_check( string str )

函数参数
参数 含义
string str 过滤前的字符串
返回值
值 含义
Boolean 存在危险参数时返回 FALSE，否则返回 TRUE

本接口函数用于检测从外部传入的参数，存在危险参数时返回 FALSE，否则返回 TRUE
