扫描模块，缓存所有Hock

array Module :: hooks_push( Boolean debug = false )

函数参数
参数 含义
Boolean debug 是否开启调试
返回值
值 含义
Array 已缓存的 Hock 列表（同时保存在全局变量 $HOCKERS 中）

扫描相应的钩子文件，存为数组，并将扫描到的信息写入Hock缓存中

遍历Hock示例 (PHP)
//遍历 Hock
Module :: hooks_push( true );

//调试 Hock
var_dump( $HOCKERS );

输出当前 Hock

string Module :: hooks_slice( string key )

函数参数
参数 含义
string key Hock 索引键名
返回值
值 含义
string 当前选择菜单的内容

将扫描结果中缓存的hock数组进行遍历并输出

输出Hock示例 (PHP)
echo Module :: hooks_slice( $key );

调用当前模块配置项

string Module :: hooks_quick( string appid )

函数参数
参数 含义
string appid 模块名称
返回值
值 含义
string string [0] 系统菜单
string [1] 模块菜单

根据 appid 调用相应 hock 的模块快捷菜单，如果不 appid 则获取系统的菜单

输出当前模块快捷菜单例 (PHP)
echo Module :: hooks_quick( $appid );
