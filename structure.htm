VeryIDE 的目录结构

api/ 系统 API 请求目录
attach/ 附件数据目录，此目录及其子目录需要有可写权限
cache/ 缓存数据目录，此目录及其子目录需要有可写权限
class/ PHP 类文件目录
config/ 配置文件目录，部分文件需要有可写权限
data/ 数据目录，包括备份、安装和更新脚本，此目录及其子目录需要有可写权限
image/ 图片目录
dialog/ 对话文件
editor/ 编辑器，Kind Editor
function/ 公共函数库，包括缓存处理、邮件发送、文件上传、字符验证和系统函数
js/ javascript 脚本目录
module/ 应用程序 module 部分目录
source/ 系统资源目录，包括 IP 库、字体文件、Flash 图表、二维码组件以及节日效果等等
smarty/ Smarty 模板引擎
static/ 模板编译及静态缓存
style/ 主框架公共样式文件
theme/ 主框架公共皮肤包
index.php 服务端接口函数库接口文件
ajax.php 主框架 Ajax 请求文件
robots.txt 搜索引擎限制文件
