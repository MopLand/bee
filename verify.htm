检查IP是否在规则内

Boolean checkIP( array array, string ip="" )

函数参数
参数 含义
array array IP规则，支持array("127.0.0.1-127.0.0.255"),array("127.0.0.*"),array("127.0.0.1")
string ip 当前IP地址
返回值
值 含义
Boolean 是否通过规则

本接口函数用于根据用户的规则来判断IP是否在规则内

用户登录示例 (PHP)
$str=array("127.0.0.1-127.0.0.255");
echo checkIP($str); //true
