UBB转HTML（简单版）

string ubb_basic( string message )

函数参数
参数 含义
string message 内容
返回值
值 含义
string 用UBB函数过滤后的HTML代码

本接口函数用于过滤UBB代码，将其替换成相应的HTML

转换UBB表情

string ubb_smile( string submsg，string image，string extra )

函数参数
参数 含义
string submsg 内容块
string image 图片目录
string extra 扩展名
返回值
值 含义
string 用UBB函数过滤后的HTML代码

本接口函数用于过滤UBB表情代码，将其替换成相应的HTML代码

清除UBB代码

string ubb_clear( string content )

函数参数
参数 含义
string content 内容块
返回值
值 含义
string 清除UBB代码后的内容块

本接口函数用于清除UBB代码，将其替换成空白字符
