获取缓存存储目录

Cached :: direct( string appid )

函数参数
参数 含义
string appid 模块ID
返回值
值 含义
string 缓存存储目录

删除缓存文件

Cached :: delete( string appid, string file )

函数参数
参数 含义
string appid 模块ID
string file 缓存名称
返回值
值 含义
boolean true:删除成功
false:删除失败

载入缓存文件

Cached :: loader( string appid, string file )

函数参数
参数 含义
string appid 模块ID
string file 缓存名称，仅限 PHP 缓存
返回值
值 含义
boolean true:载入成功
false:载入失败

Cached :: script( string appid, string sql, string var, string file, string option )

函数参数
参数 含义

缓存页面查询

string Cached :: query( string sql, string time, string temp[, bool debug] )

函数参数
参数 含义
string sql 查询语句
string time 缓存有效期，分钟
string temp 缓存模板
bool debug true:调试
false:不调试(默认)
返回值
值 含义
string cache 生成的缓存

缓存表单结构

string Cached :: form( string appid, string list )

函数参数
参数 含义
string appid 模块 ID
string list 需要缓存的数据列表
