以下关于文件的阐述以及代码的样例，均以 PHP 程序为例。

VeryIDE 配置文件

目前 VeryIDE 的全局配置文件共有 个，均位于 config/ 中：
attach.xml 远程附件和水印配置，模板文件
attach.php 远程附件和水印配置，存储文件
global.xml 网站全局配置，模板文件
global.php 网站全局配置，存储文件
mail.xml 电子邮件配置，模板文件
mail.php 电子邮件配置，存储文件
config.php 数据库配置
version.php 系统版本信息

config.php

VI_BASE VeryIDE 根目录，相对地址
VI_HOST VeryIDE 根目录，绝对地址（带有 http 信息）
VI_START 安装时间，将作为部分 API 的密钥，请勿泄露
VI_DBHOST VeryIDE 数据库主机
VI_DBPORT VeryIDE 数据库主机端口
VI_DBUSER VeryIDE 数据库用户名
VI_DBPASS VeryIDE 数据库密码
VI_DBNAME VeryIDE 数据库名称
VI_DBCHARSET VeryIDE 数据库字符集
VI_DBMANPRE VeryIDE 系统数据库表前缀
VI_DBMODPRE VeryIDE 模块数据库表前缀
VI_UCS UCS编码方式
UCS-2:默认编码，Windows 常用
UCS-2BE:部分 *nux 系统使用（请在表单等用户数据出现乱码时使用）

version.php

$_G['product']
appname 产品名称
version 产品版本
version VeryIDE 版本号
build VeryIDE 最后更新日期
charset VeryIDE 语言编码
$_G['system']
base 相对地址
host 绝对地址
root 本地地址
$_G['project']
heartbeat 心跳频率（秒）
domain 站点主域名，自动获取值
queries 数据库查询次数
check 需要检查目录权限的目录
attach 附件位置
object 对象归属
home 数据库查询次数
powered 支持信息
product 产品全称
email 服务信箱
stat 统计配置
sex 性别信息
marriage 婚姻状态
weeks 周数组
months 月数组
days 天数组
state 数据状态
func 系统权限
pages 页面权限
$_G['version']
demo 演示版
free 免费版
full 商业版
only 特别版
$_G['upload']
image 允许上传的图片类型
media 允许上传的媒体类型
flash 允许上传的Flash类型
files 允许其它的文件类型

根据 VeryIDE 版本不同，以上内容也可能不尽相同
