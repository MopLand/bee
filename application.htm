VeryIDE 模块构成

VeryIDE 中模块是以独立文件夹的形式存放于 module/ 中，每个模块的名字（我们称之为 appid ）都是唯一的，其中 system 为系统模块不能执行删除和卸载操作。

VeryIDE 模块文件

config.php 模块配置文件
function.php 模块函数包
index.php 首页文件，一般用作跳转
naver.php 公共导航文件
icon.png 图标，最佳 60×60
setting.xml 配置项，模板文件
setting.php 配置项，存储文件
navigate.xml 导航配置项，模板文件
navigate.php 导航配置项，存储文件
install.sql 安装脚本
uninstall.sql 卸载脚本
content/ 公共用户展示内容（前台）
widget/ 小插件目录，详见小工具
* 其他资源文件

VeryIDE 模块配置：config.php

模块配置文件，位于 module/*/config.php ，以下相关变量：
name 模块名称，推荐4个字以内
model 模块类型
system:系统模块
module:应用模块（提供用户访问、配置和交互的内容块，可以创建快捷方式和桌面小工具）
service:后台服务（提供服务支持的功能性模块，没有用户界面，可以创建桌面小工具）
version 模块版本号
author 开发者
support 支持页面，链接地址
describe 模块描述
serve 后台服务脚本（*.js），当用户添加到桌面时执行
appid 当前模块 APPID
* 其他配置项
以上为必选项，根据模块实际情况有可能会有所增加，具体请参考每个模块目录下面的 /config.php
